  ## CSS의 Cascading에 대해 설명해 주세요.<br>
  <b>CSS</b>는 <b>Cascading Style Sheet</b>의 약자로, 계층을 가진 스타일시트이다.<br>
  여기서 계층이란 스타일시트의 적용방식이 정해진 <s>우선순위</s> 규칙들에 의해 적용이 된다는 의미이다.<br>
  정해진 규칙에는 다음과 같이 3개가 있다.<br>
  <ul>
    <li>중요도</li>
    <li>명시도</li>
    <li>코드순서</li>
  </ul>
